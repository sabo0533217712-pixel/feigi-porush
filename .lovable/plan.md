## הבעיה
בטופס קביעת תור באדמין, בבחירת סוג טיפול, מוצג ליד שם הטיפול משך ברירת מחדל (למשל "אפילציה (30 דק׳)") גם לטיפולים מסוג "משך גמיש", שבהם אין משך קבוע.

## הפתרון
בקובץ `src/pages/admin/AdminCalendar.tsx`, בשורה 1204, להציג את משך הזמן רק לטיפולים שאינם גמישים. לטיפולים גמישים להציג את שם הטיפול בלבד, עם תווית "(משך גמיש)" כדי להבהיר למשתמש.

### שינוי נקודתי
```tsx
<SelectItem key={t.id} value={t.id}>
  {t.is_variable_duration
    ? `${t.name} (משך גמיש)`
    : `${t.name} (${t.duration_minutes} דק׳)`}
</SelectItem>
```

## מה לא משתנה
- שום לוגיקה עסקית, מוטציות, בחירת משך, race conditions, או React Query.
- שינוי טקסטואלי בלבד בתצוגת ה-Select.
