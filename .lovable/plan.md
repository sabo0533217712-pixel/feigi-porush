

# שינויים ביומן האדמין

## סקירה

שכתוב מסך יומן האדמין כך שלחיצה על יום תפתח תצוגת ציר זמן (timeline) מעל היומן, עם יכולת קביעת תורים וחסימות זמן ישירות מהציר.

## שינויי מסד נתונים

**Migration חדש** — טבלת `time_blocks` לחסימות זמן:
```sql
CREATE TABLE public.time_blocks (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  block_date date NOT NULL,
  start_time time NOT NULL,
  end_time time NOT NULL,
  notes text DEFAULT '',
  created_at timestamptz DEFAULT now()
);
ALTER TABLE public.time_blocks ENABLE ROW LEVEL SECURITY;
-- RLS: admin only
```

## שינויים ב-AdminCalendar.tsx (שכתוב מלא)

### 1. יומן גדול יותר
- הגדלת תאי הימים בקלנדר (w-14 h-14 במקום w-10 h-10)
- הסרת התצוגה השבועית הישנה (הקלנדר עצמו מציג חודש שלם)

### 2. תצוגת ציר זמן מעל היומן
- בלחיצה על יום → נפתח מעל היומן (לא לצד) אזור timeline
- ציר אנכי לפי שעות (מ-start_time עד end_time של אותו יום מ-day_schedules)
- כל תור מוצג כבלוק צבעוני בגובה יחסי למשך שלו
- חסימות זמן מוצגות כבלוקים אפורים מפוספסים

### 3. קביעת תור מהציר
- לחיצה על שעה ריקה בציר → פותח dialog עם:
  - שעת התחלה (מוגדרת אוטומטית לפי השעה שנלחצה)
  - שעת סיום (מתעדכנת אוטומטית לפי הטיפול)
  - בחירת טיפול
  - בחירת לקוחה מרשימה
  - הערות
- האדמין יכולה לקבוע גם בזמן חסום / מחוץ לשעות / חופף

### 4. חסימת זמן
- כפתור "חסימת זמן" בציר → dialog עם שעת התחלה וסיום בלבד + הערה אופציונלית
- החסימה נשמרת ב-`time_blocks` ומוצגת בציר כבלוק אפור
- הלקוחות לא יוכלו להזמין בזמן חסום (עדכון ב-ClientBooking)

### 5. עריכת תור קיים
- לחיצה על תור בציר → dialog עריכה שמאפשר שינוי:
  - שעת התחלה / סיום (ידני)
  - סטטוס
  - הערות
- שמירה עם `supabase.from('appointments').update(...)`

### 6. כפתורי תקשורת + פרטי לקוחה
- לכל תור בציר: אייקוני מייל, SMS, טלפון
- כפתור "פרטי לקוחה" שמציג popup עם שם, טלפון, מייל

### 7. חופש מלא לאדמין
- `booked_by_admin: true` → עוקף כל בדיקת חפיפה/שעות/חסימה (כבר קיים ב-trigger)
- עדכון ClientBooking לסנן גם `time_blocks` מהזמנים הפנויים
