

# הסרת כפתורי עריכה/פרטים מהתור + לחיצה על תור פותחת עריכה

## שינויים בקובץ `AdminCalendar.tsx`

### 1. הסרת כפתורי User ו-Edit מכרטיס התור
הסרת ה-div שמכיל את שני הכפתורים (שורות 546-556) — כפתור פרטי לקוחה וכפתור עריכה.

### 2. לחיצה על כרטיס התור פותחת חלונית עריכה
הוספת `onClick={() => openEditDialog(apt)}` על ה-div הראשי של כרטיס התור (שורה ~526), עם `cursor-pointer`.

### 3. הוספת כפתור "יצירת קשר" בתוך חלונית העריכה
בחלונית עריכת תור (שורות 794-819), הוספת כפתור "פרטי לקוחה / יצירת קשר" שפותח את חלונית ה-Client Info הקיימת:
```
<Button variant="outline" size="sm" onClick={() => setShowClientInfo(editingAppointment)}>
  <User /> יצירת קשר
</Button>
```

הכפתור יוצב מתחת לאזור פרטי הלקוח בחלונית העריכה.
